Bulk Member Import

Import existing customers as member accounts. Members receive their Member ID by email and set their password via Forgot Password.
Go to Website ← Dashboard

Instructions

  • Download the sample Excel, fill in customer details, and upload the completed file.
  • Required column: Name. At least one of Email or Mobile Number is required per row.
  • Created At preserves the original registration date when provided (e.g. 01 Feb 2025, 08:28 AM).
  • Passwords are not created or emailed. Members use Forgot Password on first login.
  • Duplicate members (existing email or mobile) are skipped automatically.
  • After import, download the error report to correct and re-upload failed rows only.
